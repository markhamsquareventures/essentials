# Markham Square essentials

An opinionated and tailored collection of Claude skills, commands, and hooks for the Markham Square team.

Focused on working with Laravel projects.

## Installation

In Claude Code, register the marketplace first:

`/plugin marketplace add markhamsquareventures/mksq-marketplace`

Then install the plugin from this marketplace:

`/plugin install essentials@mksq-marketplace`
